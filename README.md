# Interactive Sensor System
Author: **Ecem Karaduman** · GitHub: **@ecemkrd**

A simple interactive sensor pipeline using Python, WebSockets, and a browser visualizer.
Shows real‑time mapping of sensor data to visual behavior.

## Structure
- `software/` — server + visualizer
- `hardware/` — parts list
- `docs/` — optional notes

## Run
```
python software/server.py
```
Open `visualizer.html` in your browser.
